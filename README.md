# 🎓 Student Placement Prediction

An End-to-End Machine Learning project that predicts whether a student is likely to get placed based on their academic performance.

The prediction is made using the following features:
- 📘 CGPA
- 🏫 10th Percentage
- 🏫 12th Percentage

The project includes data preprocessing, model training, model evaluation, a Flask web application, and deployment.
